technologies used:<br/>
react native <br/>
firebase cloud firestore <br/>

features: <br/>
anonymous chat <br/>

reply to texts like in twitter <br/>
![app screenshot](appshot3.png?raw=true "screenshot")  <br/>

![app screenshot](appshot1.png?raw=true "screenshot")  <br/>
